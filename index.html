Heterograma, Isograma y Pangrama

Las siguientes funciones permiten determinar si un string (cadena de texto) es un Heterograma, Isograma o Pangrama

Heterograma

Un heterograma es una cadena de texto o string, bien sea una palabra o frase en donde no se repiten las letras o donde no existen letras repetidas
